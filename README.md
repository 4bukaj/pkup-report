How to run it:

1. Clone the repository into the same root folder your repo that you want to pull the commits from is.
2. Create .env file with following keys:
   OPENAI_API_KEY
   ORG_ID - atlasian organization ID
   PROJECT_ID - Jira project ID
   ATLASIAN_API_KEY
   USER_EMAIL - your email from git config
3. Run it with npm start

You'll get commits folder and output.txt for pkup report.
